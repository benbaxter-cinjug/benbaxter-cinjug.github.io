From Java to JavaScript March 2015

Part 1: The Build Process

View Prensentation
Live in the land of web applications and love writing Java code that does amazing things? What do you do when pesky IE 8 doesn't let you play with the shinier technology? You tuck your tail between your legs and spend the rest of the week patching JavaScript to adhere to Bill Gate's rules! Still have to support IE 6 and have become accustomed to seeing new tools but then your spirits gets crushed because they only work with IE 30 and up? Well good news! There are some really cool JavaScript libraries that let you enjoy the 'joys' of JavaScript without the hassle of the browser. I am referring to the new world that Node.js has opened for us. We will look at gulp.js and examine how to setup a build system in JavaScript for JavaScript. We will be able to mini-fy, ugly-fy, obfuscate-fy, and french-fy our JavaScript in a self contained way! We will briefly discuss node and bower and the parallels between Java and these new shiny technologies. Now we can finally incorporate 'optimize' client side code to squeeze out a little bit more performance that we have always neglected. Even if you have to support 'legacy' versions of IE, you can still get some useful tricks and tips from this talk! I hope to see you there!

Spring Boot July 2014

View Presentation
We are going to go on an adventure that no magician would allow, as we reveal a few secrets tricks. We will stroll down memory lane and laugh at how naive we, oh so many year ago, thought that xml configuration was the latest and greatest craze. Spring Boot is the next step in the spring ecosystem that will remove even more boiler plate code and make bootstrapping projects a breeze. We will be building a simple guestbook application and slowly make it look more and more like a real grown up app than just some sample 'look what I can do' application. This journey will make those who love spring and those who are just starting out to never stop believing in the magic of spring.
